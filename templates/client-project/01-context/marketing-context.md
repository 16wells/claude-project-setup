# Marketing Context — {{CLIENT_NAME}}

> **Positioning and messaging foundation for marketing-led engagements.** This file is where the *what we say, to whom, and why* lives — separate from `client-profile.md`, which covers *who they are.* When drafting any marketing content (ads, landing pages, emails, social, blog posts), read this first.

## When to Use This File

This file is **optional.** Create and fill it when the engagement is marketing-led — campaigns, content production, ad strategy, positioning work, rebrands. For project types where marketing context is peripheral (say, a pure analytics audit or a technical implementation), you can skip it.

If this file exists, treat it as the single source of truth for voice, positioning, and messaging. Other content-drafting work should reference it rather than re-deriving positioning from scratch.

## How to Build It

- **From the client's website and materials** — scan the public site, About page, service/product pages, existing ads, and sales collateral.
- **From the user's notes** — kickoff call, proposal, competitive analysis, any discovery work.
- **From reviews and testimonials** — customer language, pain points, and value themes surface here.
- **Never invent.** If a section is empty after a real look, leave it empty with a `TODO:` note rather than making it up. Half-filled is more useful than fabricated.

Update this file as new insights surface during the engagement.
